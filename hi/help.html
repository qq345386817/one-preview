OnePreview Help

Extension enable करने और preview behavior समझने के लिए यह guide पढ़ें.

macOS पर OnePreview कैसे enable करें?

Install के बाद OnePreview एक बार खोलें. macOS Sequoia में System Settings > General > Login Items & Extensions > Quick Look पर जाकर OnePreview enable करें. macOS Sonoma में System Settings > Privacy & Security > Extensions > Quick Look का उपयोग करें.

iPhone या iPad पर कैसे उपयोग करें?

OnePreview install करके एक बार खोलें. फिर Files app या किसी system Quick Look app में supported file खोलें.

कुछ local Markdown images क्यों नहीं दिखतीं?

Apple sandbox rules Quick Look extension को related local files पढ़ने से रोक सकते हैं. Web image URLs सबसे भरोसेमंद हैं.

Trash में files preview क्यों नहीं होतीं?

macOS third-party Quick Look extensions को Trash में मौजूद items पढ़ने से रोकता है. File को Trash से बाहर ले जाएं.

क्या OnePreview files edit करता है?

नहीं. OnePreview सिर्फ viewer है और files modify नहीं करता.
