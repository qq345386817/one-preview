Markdown, code, data और folders के लिए Quick Look.

OnePreview Mac, iPhone और iPad पर Markdown, JSON, JSONL, CSV, source code, plain text और folders के लिए Apple Quick Look को बेहतर बनाता है.

App Store पर देखें Mac DMG डाउनलोड करें मदद पढ़ें 
# OnePreview

```swift
let file = QuickLook.preview("demo.jsonl")
```

| Format | Preview |
| --- | --- |
| Markdown | Rendered + Source |
| JSONL | Formatted records |
| CSV | Table |

App खोले बिना file समझें

Markdown साफ दिखता है

GitHub-flavored Markdown, code highlighting, tables, math, Mermaid diagrams, alerts और source mode.

Structured data जल्दी पढ़ें

JSON, JSONL, YAML, XML, CSV और TSV तेज inspection के लिए format होते हैं.

Source code context के साथ

Swift, Python, JavaScript, C++, Rust, Go, Ruby, Java, shell scripts और अधिक.

Folder previews

Quick Look से folder contents, file sizes और modification dates देखें.

Supported formats

Markdown, JSON, JSONL, YAML, XML, CSV, TSV, Swift, Python, JavaScript, TypeScript, C, C++, Rust, Go, Ruby, Java, shell scripts, plain text, logs और folders.

Developers की file browsing आदतों के लिए बनाया गया

  1. macOS Finder में Spacebar दबाएं, या iPhone/iPad पर Files app में Quick Look खोलें.
  2. Markdown, JSON और JSONL में rendered preview और raw source के बीच switch करें.
  3. हर छोटी जांच के लिए full editor खोलने की जरूरत नहीं.
