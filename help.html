OnePreview Help

Use this guide to enable the extension and understand the main preview behaviors.

How do I enable OnePreview on macOS?

Open OnePreview once after installation. On macOS Sequoia, go to System Settings > General > Login Items & Extensions > Quick Look and enable OnePreview. On macOS Sonoma, use System Settings > Privacy & Security > Extensions > Quick Look.

How do I use it on iPhone or iPad?

Install and open OnePreview once. Then open a supported file in Files or any app that uses system Quick Look.

Why do some local Markdown images not appear?

Apple sandbox rules limit what a Quick Look extension can read. Web image URLs are the most reliable. The separately distributed DMG build can allow broader local image access where permitted.

Why does preview not work in Trash?

macOS blocks third-party Quick Look extensions from reading items in Trash. Move the file out of Trash before previewing it.

Does OnePreview edit my files?

No. OnePreview is a viewer. It does not modify your files.
