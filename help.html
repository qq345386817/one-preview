OnePreview FAQ

OnePreview is a powerful Quick Look enhancement tool for the Apple ecosystem. It allows you to instantly preview files that are not natively supported (like Markdown, Swift, JSON, CSV) simply by pressing the spacebar, without opening any dedicated app.

1. Open the "OnePreview" app once to register the extension with macOS.

2. In Finder, select a supported file (e.g., a `.md` or `.swift` file) and simply press the Spacebar to see the rich preview.

No. After you launch the app for the first time on macOS (which is required to register the extension), you can close it. The preview extension operates seamlessly in the background.

• Markdown: Beautifully rendered HTML and source syntax highlighting.

• Source Code: Swift, Python, JS, C++, Shell, etc.

• Structured Data: JSON, YAML, XML with syntax colors, and CSV/TSV rendered as neat tables.

• Folders: Preview the contents inside a directory, including file sizes and modification dates.

macOS Quick Look occasionally caches previews. You can close the preview window (press Spacebar again) and reopen it to force a refresh.

Due to Apple's strict sandbox security policies, extensions sometimes face restrictions reading local files referenced inside documents. We employ techniques to bypass this for adjacent files when possible, but remote web URLs (`https://...`) are generally the most reliable for images.
