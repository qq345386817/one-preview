OnePreview

Markdown, 코드, 데이터, 폴더를 위한 Quick Look.

OnePreview는 Mac, iPhone, iPad의 Apple Quick Look을 확장하여 Markdown, JSON, JSONL, CSV, 소스 코드, 일반 텍스트, 폴더를 더 선명하게 미리 보여줍니다.

App Store에서 보기 Mac DMG 다운로드 도움말 읽기 
# OnePreview

```swift
let file = QuickLook.preview("demo.jsonl")
```

| Format | Preview |
| --- | --- |
| Markdown | Rendered + Source |
| JSONL | Formatted records |
| CSV | Table |

앱을 열기 전에 파일을 파악

읽기 쉬운 Markdown

GitHub Flavored Markdown, 코드 하이라이트, 표, 수식, Mermaid 다이어그램, 알림, 소스 모드를 지원합니다.

구조화 데이터를 빠르게 확인

JSON, JSONL, YAML, XML, CSV, TSV를 빠른 점검에 맞게 표시합니다.

맥락이 보이는 소스 코드

Swift, Python, JavaScript, C++, Rust, Go, Ruby, Java, 셸 스크립트 등을 하이라이트합니다.

폴더 미리보기

폴더 내용, 파일 크기, 수정 시간을 Quick Look에서 바로 확인합니다.

지원 형식

Markdown, JSON, JSONL, YAML, XML, CSV, TSV, Swift, Python, JavaScript, TypeScript, C, C++, Rust, Go, Ruby, Java, 셸 스크립트, 일반 텍스트, 로그, 폴더.

개발자의 파일 탐색 방식에 맞춘 설계

  1. macOS Finder에서는 스페이스바를 누르고, iPhone 및 iPad에서는 파일 앱이나 Quick Look 지원 앱에서 엽니다.
  2. Markdown, JSON, JSONL은 렌더링 보기와 원본 소스를 전환할 수 있습니다.
  3. 간단한 확인마다 전체 편집기를 열 필요가 없습니다.
