OnePreview

Quick Look untuk Markdown, kode, data, dan folder.

OnePreview memperluas Apple Quick Look di Mac, iPhone, dan iPad dengan pratinjau rapi untuk Markdown, JSON, JSONL, CSV, kode sumber, teks biasa, dan folder.

Lihat di App Store Unduh DMG Mac Baca Bantuan 
# OnePreview

```swift
let file = QuickLook.preview("demo.jsonl")
```

| Format | Preview |
| --- | --- |
| Markdown | Rendered + Source |
| JSONL | Formatted records |
| CSV | Table |

Pahami file sebelum membuka aplikasi

Markdown yang mudah dibaca

Render Markdown gaya GitHub dengan sorotan kode, tabel, matematika, diagram Mermaid, alert, dan mode sumber.

Data terstruktur sekali lihat

JSON, JSONL, YAML, XML, CSV, dan TSV ditampilkan untuk inspeksi cepat.

Kode sumber dengan konteks

Sorotan untuk Swift, Python, JavaScript, C++, Rust, Go, Ruby, Java, skrip shell, dan lainnya.

Pratinjau folder

Lihat isi folder, ukuran file, dan tanggal perubahan langsung dari Quick Look.

Format yang didukung

Markdown, JSON, JSONL, YAML, XML, CSV, TSV, Swift, Python, JavaScript, TypeScript, C, C++, Rust, Go, Ruby, Java, skrip shell, teks biasa, log, dan folder.

Dibuat untuk cara developer menelusuri file

  1. Tekan Spacebar di Finder pada macOS, atau buka Quick Look dari Files di iPhone dan iPad.
  2. Beralih antara pratinjau render dan sumber mentah untuk Markdown, JSON, dan JSONL.
  3. Tetap di alur pratinjau sistem tanpa membuka editor penuh untuk setiap pengecekan cepat.
