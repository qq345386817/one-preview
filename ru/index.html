Quick Look для Markdown, кода, данных и папок.

OnePreview расширяет Apple Quick Look на Mac, iPhone и iPad, добавляя понятный предпросмотр Markdown, JSON, JSONL, CSV, исходного кода, простого текста и папок.

| Format | Preview |
| --- | --- |
| Markdown | Rendered + Source |
| JSONL | Formatted records |
| CSV | Table |

Понимайте файл до открытия приложения

Markdown читается сразу

GitHub Flavored Markdown с подсветкой кода, таблицами, формулами, Mermaid-диаграммами, alert-блоками и режимом исходника.

Структурированные данные с первого взгляда

JSON, JSONL, YAML, XML, CSV и TSV форматируются для быстрой проверки.

Исходный код с контекстом

Подсветка для Swift, Python, JavaScript, C++, Rust, Go, Ruby, Java, shell-скриптов и других языков.

Предпросмотр папок

Смотрите содержимое папки, размеры файлов и даты изменения прямо из Quick Look.

Поддерживаемые форматы

Markdown, JSON, JSONL, YAML, XML, CSV, TSV, Swift, Python, JavaScript, TypeScript, C, C++, Rust, Go, Ruby, Java, shell-скрипты, простой текст, логи и папки.

Создано для разработчиков, которые быстро просматривают файлы

  1. Нажмите пробел в Finder на macOS или откройте Quick Look из Files на iPhone и iPad.
  2. Переключайтесь между отрисованным предпросмотром и исходным текстом для Markdown, JSON и JSONL.
  3. Оставайтесь в системном предпросмотре вместо открытия полноценного редактора для каждой проверки.
