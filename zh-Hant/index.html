OnePreview 擴展 Apple 系統快速查看能力，在 Mac、iPhone 和 iPad 上為 Markdown、JSON、JSONL、CSV、原始碼、純文字和資料夾提供更清晰的預覽。

不用打開 App，先看懂檔案

Markdown 直接排版

支援 GitHub 風格 Markdown、程式碼高亮、表格、數學公式、Mermaid 圖表、提示框和原始碼模式。

結構化資料一眼讀懂

JSON、JSONL、YAML、XML、CSV、TSV 都能以適合快速檢查的方式呈現。

原始碼高亮有脈絡

支援 Swift、Python、JavaScript、C++、Rust、Go、Ruby、Java、Shell 等主流語言。

資料夾也能預覽

直接查看資料夾內容、大小和修改時間，不必先進入目錄。

支援格式

Markdown、JSON、JSONL、YAML、XML、CSV、TSV、Swift、Python、JavaScript、TypeScript、C、C++、Rust、Go、Ruby、Java、Shell、純文字、日誌和資料夾。

為開發者瀏覽檔案的習慣而生

  1. macOS 在 Finder 中按空白鍵；iPhone 和 iPad 在檔案 App 或支援快速查看的 App 中打開。
  2. Markdown、JSON、JSONL 可在渲染視圖和原始文字之間切換。
  3. 不必為了確認一個檔案而打開完整編輯器，保持在系統預覽流程裡完成判斷。
