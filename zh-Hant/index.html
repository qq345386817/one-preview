打開編輯器之前，先預覽 Markdown、程式碼、資料和 ZIP 壓縮檔。

OnePreview 讓檔案檢查留在 Finder、檔案 App 和系統預覽流程裡。按一下空白鍵，就能閱讀文件、檢查結構化資料、瀏覽壓縮檔和查看原始碼。

覆蓋日常檢查的主要檔案類型

Markdown 與文件

GitHub 風格 Markdown、表格、程式碼區塊、數學公式、Mermaid 圖表、提示區塊、純文字、日誌和文件檔案。

結構化資料

JSON、JSONL、YAML、XML、CSV、TSV、TOML、ENV、Apple plist 等文字資料格式。

原始碼檔案

Swift、Python、JavaScript、TypeScript、C、C++、Objective-C、Rust、Go、Ruby、Java、Shell、Assembly 等。

壓縮檔與資料夾

ZIP 壓縮檔內容、資料夾列表、檔案大小、修改時間和 SQLite 資料庫概覽。

Technical formats: Markdown, JSON, JSONL, YAML, XML, CSV, TSV, TOML, ENV, SQLite, Swift, Python, JavaScript, TypeScript, C, C++, Objective-C, Rust, Go, Ruby, Java, Shell, Assembly, source code, plain text, logs, folders, ZIP archives, Apple property lists (plist).
