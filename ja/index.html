オールインワン Quick Look 拡張

OnePreview

Markdown、コード、データ、フォルダのための Quick Look。

OnePreview は Mac、iPhone、iPad の Apple Quick Look を拡張し、Markdown、JSON、JSONL、CSV、ソースコード、プレーンテキスト、フォルダを見やすくプレビューします。

App Store で見る Mac DMG をダウンロード ヘルプを読む 
# OnePreview

```swift
let file = QuickLook.preview("demo.jsonl")
```

| Format | Preview |
| --- | --- |
| Markdown | Rendered + Source |
| JSONL | Formatted records |
| CSV | Table |

アプリを開く前にファイルを把握

読みやすい Markdown

GitHub Flavored Markdown、コードハイライト、表、数式、Mermaid 図、アラート、ソース表示に対応。

構造化データをすばやく確認

JSON、JSONL、YAML、XML、CSV、TSV を点検しやすい形で表示します。

文脈が見えるソースコード

Swift、Python、JavaScript、C++、Rust、Go、Ruby、Java、シェルスクリプトなどをハイライト。

フォルダもプレビュー

フォルダ内の項目、サイズ、更新日時を Quick Look から直接確認できます。

対応形式

Markdown、JSON、JSONL、YAML、XML、CSV、TSV、Swift、Python、JavaScript、TypeScript、C、C++、Rust、Go、Ruby、Java、シェルスクリプト、プレーンテキスト、ログ、フォルダ。

開発者のファイル確認に合わせた設計

  1. macOS では Finder でスペースバー、iPhone と iPad ではファイル App などから Quick Look を開きます。
  2. Markdown、JSON、JSONL はレンダリング表示と生のソースを切り替えられます。
  3. ちょっとした確認のたびにフル機能のエディタを開く必要がありません。
