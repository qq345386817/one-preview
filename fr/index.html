Quick Look pour Markdown, le code, les données et les dossiers.

OnePreview enrichit Apple Quick Look sur Mac, iPhone et iPad avec des aperçus lisibles pour Markdown, JSON, JSONL, CSV, code source, texte brut et dossiers.

Comprendre le fichier avant d’ouvrir une app

Markdown lisible

Markdown GitHub avec coloration du code, tableaux, formules, diagrammes Mermaid, alertes et mode source.

Données structurées d’un coup d’œil

JSON, JSONL, YAML, XML, CSV et TSV sont formatés pour une inspection rapide.

Code source avec contexte

Coloration pour Swift, Python, JavaScript, C++, Rust, Go, Ruby, Java, scripts shell et plus encore.

Aperçu des dossiers

Consultez le contenu, les tailles et les dates de modification des dossiers depuis Quick Look.

Formats pris en charge

Markdown, JSON, JSONL, YAML, XML, CSV, TSV, Swift, Python, JavaScript, TypeScript, C, C++, Rust, Go, Ruby, Java, scripts shell, texte brut, journaux et dossiers.

Conçu pour la navigation des développeurs

  1. Appuyez sur Espace dans le Finder sur macOS, ou ouvrez Quick Look depuis Fichiers sur iPhone et iPad.
  2. Passez de l’aperçu rendu au source brut pour Markdown, JSON et JSONL.
  3. Restez dans le flux d’aperçu système au lieu d’ouvrir un éditeur complet pour chaque vérification.
