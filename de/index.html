OnePreview

Quick Look für Markdown, Code, Daten und Ordner.

OnePreview erweitert Apple Quick Look auf Mac, iPhone und iPad mit klaren Vorschauen für Markdown, JSON, JSONL, CSV, Quellcode, einfachen Text und Ordner.

Im App Store ansehen Mac-DMG laden Hilfe lesen 
# OnePreview

```swift
let file = QuickLook.preview("demo.jsonl")
```

| Format | Preview |
| --- | --- |
| Markdown | Rendered + Source |
| JSONL | Formatted records |
| CSV | Table |

Dateien verstehen, bevor du eine App öffnest

Markdown klar gelesen

GitHub-Flavored Markdown mit Code-Highlighting, Tabellen, Formeln, Mermaid-Diagrammen, Alerts und Quellmodus.

Strukturierte Daten auf einen Blick

JSON, JSONL, YAML, XML, CSV und TSV werden für schnelle Prüfung übersichtlich formatiert.

Quellcode mit Kontext

Highlighting für Swift, Python, JavaScript, C++, Rust, Go, Ruby, Java, Shell-Skripte und mehr.

Ordnervorschau

Ordnerinhalte, Dateigrößen und Änderungsdaten direkt in Quick Look prüfen.

Unterstützte Formate

Markdown, JSON, JSONL, YAML, XML, CSV, TSV, Swift, Python, JavaScript, TypeScript, C, C++, Rust, Go, Ruby, Java, Shell-Skripte, Text, Logs und Ordner.

Für die Dateiarbeit von Entwicklern gebaut

  1. Drücke unter macOS im Finder die Leertaste oder öffne Quick Look in Dateien auf iPhone und iPad.
  2. Wechsle bei Markdown, JSON und JSONL zwischen gerenderter Vorschau und Rohtext.
  3. Bleib im Systemvorschau-Fluss, statt für jede kurze Prüfung einen Editor zu öffnen.
