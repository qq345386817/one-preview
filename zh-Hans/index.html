为 Markdown、代码、数据和文件夹增强快速查看。

OnePreview 扩展 Apple 系统快速查看能力，在 Mac、iPhone 和 iPad 上为 Markdown、JSON、JSONL、CSV、源码、纯文本和文件夹提供更清晰的预览。

# OnePreview

```swift
let file = QuickLook.preview("demo.jsonl")
```

| Format | Preview |
| --- | --- |
| Markdown | Rendered + Source |
| JSONL | Formatted records |
| CSV | Table |

不用打开应用，先看懂文件

Markdown 直接排版

支持 GitHub 风格 Markdown、代码高亮、表格、数学公式、Mermaid 图表、提示框和源码模式。

结构化数据一眼读懂

JSON、JSONL、YAML、XML、CSV、TSV 都能以适合快速检查的方式呈现。

源码高亮有上下文

支持 Swift、Python、JavaScript、C++、Rust、Go、Ruby、Java、Shell 等主流语言。

文件夹也能预览

直接查看文件夹内容、大小和修改时间，不必先进入目录。

支持格式

Markdown、JSON、JSONL、YAML、XML、CSV、TSV、Swift、Python、JavaScript、TypeScript、C、C++、Rust、Go、Ruby、Java、Shell、纯文本、日志和文件夹。

为开发者浏览文件的习惯而生

  1. macOS 在 Finder 中按空格键；iPhone 和 iPad 在文件 App 或支持快速查看的 App 中打开。
  2. Markdown、JSON、JSONL 可在渲染视图和原始文本之间切换。
  3. 不必为了确认一个文件而打开完整编辑器，保持在系统预览流程里完成判断。
