Quick Look for Markdown, code, data, and folders.

See the file before opening the app

Markdown that reads cleanly

Render GitHub-flavored Markdown with code highlighting, tables, math, Mermaid diagrams, alerts, and source mode.

Structured data at a glance

Preview JSON, JSONL, YAML, XML, CSV, and TSV with formatting built for fast inspection.

Source code with context

Highlight Swift, Python, JavaScript, C++, Rust, Go, Ruby, Java, shell scripts, and more.

Folder previews

Inspect folder contents, file sizes, and modification dates directly from Quick Look.

Supported formats

Markdown, JSON, JSONL, YAML, XML, CSV, TSV, Swift, Python, JavaScript, TypeScript, C, C++, Rust, Go, Ruby, Java, shell scripts, plain text, logs, and folders.

Built for the way developers browse files

  1. Press Spacebar in Finder on macOS, or open Quick Look from Files on iPhone and iPad.
  2. Switch between rendered preview and raw source for Markdown, JSON, and JSONL.
  3. Stay inside the system preview flow instead of opening a full editor for every quick check.
